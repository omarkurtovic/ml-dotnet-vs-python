# ML .NET vs Python

A web app for training and comparing machine learning models, built with .NET 10.

## Overview

The app lets you train, view, and delete machine learning models for two use cases:
- **Sentiment Analysis**
- **Lung Cancer Detection**

## Tech Stack

- **Frontend:** Blazor (.NET 10)
- **C# API:** ASP.NET Core / ML.NET
- **Python API:** FastAPI / scikit-learn / keras
- **Infrastructure:** HETZNER
